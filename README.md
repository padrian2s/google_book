# Google Book

Interactive study guide and simulators for *How Google Works*.

## Overview

A web-based reader with interactive simulators that bring the concepts from the book to life. Includes chapter-by-chapter navigation and hands-on exercises.

## Getting Started

```bash
./run.sh
```

Open `index.html` in your browser, or use `reader.html` for the reading view. Explore the `simulators/` directory for interactive demos.
